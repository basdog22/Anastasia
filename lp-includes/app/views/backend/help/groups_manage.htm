User Groups

This is where you manage your user groups. There are five default user groups:

  • :
    This is the group with the owners of the system. They can do everything
  • :
    This is the group with the Administrators of the system. They can use most of the available features.
  • :
    Editors are users that manage the workflow. They can publish and unpublish content and manage content revisions. They can also edit assets info.
  • :
    Authors are able to create content. They cannot publish it as it is editor's job. They can use the filemanager too.
  • :
    Members are typical registered users. They have access to all public and specific areas of the system.

You can create more user groups if you want to.
