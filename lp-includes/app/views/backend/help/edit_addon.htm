Plugin Editor

You can use this editor to make changes to any of your plugins’ individual files. Be aware that if you make changes, plugins updates will overwrite your customizations.
Choose a folder to edit from the menu in the upper right by clicking on it's name. Click once on any file name to load it in the editor, and make your changes. Don’t forget to save your changes (Save) when you’re finished.
The editor is powerfull enough to understand and highlight the code in the correct way.
